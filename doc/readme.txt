ключ -random запускает генератор случайных сэмплов вместо usrp-адаптера. скорость генерации ~ 5 с/мсек. размер буфера - 512 

опции config.ini файла, которых нет в интерфейсе:
fftw_window - размер окна БПФ. по умолчанию - 8192
fftw_overlap - размер перекрытия (число сэмплов, не изымаемое из входного потока при заполнении окна). должно быть меньше fftw_window. по умолчанию - 0
